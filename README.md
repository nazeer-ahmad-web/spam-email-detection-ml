# Spam Email Detection using Machine Learning

This project classifies emails as Spam or Not Spam using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)
- Naive Bayes

## Dataset
SMS Spam Collection Dataset from Kaggle.

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the project:
   python spam_classifier.py

## Output
Achieves over 95% accuracy in spam detection.
